Ubuntu LC_ALL = (unset)

Tags: Ubuntu LC_ALL

Posted on 2015-04-09

При установке Ubuntu (в основном на VDS) периодический вылезает ошибка:

LANGUAGE = (unset)
LC_ALL = (unset)

Для исправления делаем:

# echo LC_ALL="ru_RU.UTF-8" >> /etc/environment
# locale-gen ru_RU.UTF-8

И перелегониваемся.
